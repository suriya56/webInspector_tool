🤖 AI UI Editor Bridge - Test Page

How to test:
  1. Open this page in Chrome/Edge
  2. Open DevTools Console (F12)
  3. Paste the contents of inject.js into the console
  4. Or run: node load-toolbar.js to inject automatically
  5. The floating UI should appear in the top-right corner
  6. Click "Connect" to connect to the FastAPI bridge
  7. Click "Start Inspecting" and hover over elements
  8. Click an element to select it
  9. Enter a prompt and click "Apply Changes"
